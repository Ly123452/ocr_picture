# BOSS直聘信息识别工具

自动识别BOSS直聘软件界面中的重要信息，包括左侧候选人列表、聊天消息内容、未读状态等。

## 功能特性

- **左侧候选人列表识别**：自动检测候选人卡片，识别未读/已读状态
- **聊天消息内容识别**：识别中间聊天区域的对话内容
- **个人信息提取**：提取聊天对象的姓名和职位信息
- **OCR文字识别**：使用RapidOCR引擎识别图片中的文字内容
- **结构化输出**：生成JSON和TXT两种格式的报告

## 环境要求

- Python 3.12+
- Windows操作系统
- 已安装必要的Python包（见下方安装步骤）

## 安装依赖

### 1. 安装Python

确保已安装Python 3.12或更高版本。可从 [Python官网](https://www.python.org/) 下载安装。

### 2. 安装依赖包

打开命令行，运行以下命令安装所需依赖：

```bash
py -3.12 -m pip install opencv-python numpy Pillow rapidocr-onnxruntime -i https://pypi.tuna.tsinghua.edu.cn/simple
```

或者使用项目中的 `requirements.txt` 文件：

```bash
py -3.12 -m pip install -r requirements.txt
```

## 使用方法

### 准备截图

1. 打开BOSS直聘软件
2. 切换到需要识别的聊天窗口
3. 截取包含左侧候选人列表和中间聊天区域的屏幕截图
4. 将截图保存到本地任意位置

### 运行识别程序

```bash
py -3.12 ocr_recognizer.py
```

### 输入图片路径

程序运行后会提示你输入图片路径：

```
============================================================
BOSS直聘信息识别工具
============================================================

请输入截图文件的完整路径
提示：可以直接拖拽图片文件到命令行窗口
（直接按回车使用默认路径: mmexport1781002649103.jpg）

图片路径: 
```

你可以：
- **直接按回车**：使用默认路径 `mmexport1781002649103.jpg`
- **输入完整路径**：例如 `D:/screenshots/boss01.jpg`
- **拖拽文件**：直接将图片文件拖拽到命令行窗口，路径会自动填入

## 识别内容说明

### 左侧候选人列表

- **候选人总数**：列表中的候选人数量
- **未读消息**：带有红色标记的未读候选人数量
- **卡片详情**：每个候选人的姓名和最新消息预览

### 聊天区域消息

- **消息总数**：聊天记录中的消息数量
- **我方消息**：你自己发送的消息
- **对方消息**：候选人/老板发送的消息
- **消息内容**：OCR识别出的具体文字内容

## 常见问题

### Q: 程序运行时报错 "No module named 'xxx'"
A: 请确保已正确安装所有依赖包。重新运行安装命令：
```bash
py -3.12 -m pip install opencv-python numpy Pillow rapidocr-onnxruntime
```

### Q: OCR识别结果为空或不准
A: 请确保截图清晰，包含完整的聊天区域。可以尝试：
1. 调整截图范围，确保文字区域完整
2. 检查图片是否清晰，没有过多的模糊或阴影

### Q: 如何识别其他软件界面？
A: 当前程序针对BOSS直聘界面进行了优化。如需识别其他软件，可以：
1. 根据目标软件的界面调整颜色阈值
2. 修改卡片区域和聊天区域的提取参数

## 注意事项

1. **截图要求**：截图应包含完整的左侧候选人列表和中间聊天区域
2. **图片格式**：支持JPG、PNG等常见图片格式
3. **首次运行**：RapidOCR首次运行时会下载模型文件，请保持网络连接
4. **识别精度**：OCR识别精度受图片质量影响，请确保截图清晰
